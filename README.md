# 医院预约挂号系统
主要技术栈版本：<br>
前端 ：Vue3.x+NUXT框架+Element-UI+EChars
<br>
后台 ：Springboot+SprngCloudAlibaba微服务组件+Redis+Mybatis-Plus
<br>
数据库 ：MongoDB+MySQL8.0
<br>
分布式调度中心 : xxl-job
<br>
注册中心 ：Nacos
<br>
API接口生成 ：swagger3.x
